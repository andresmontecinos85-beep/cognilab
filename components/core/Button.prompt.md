Pill-shaped primary action; use for CTAs like booking, contacting, or navigating between key views.

```jsx
<Button variant="primary" href="#contacto">Agenda tu hora</Button>
<Button variant="whatsapp" leadingIcon={<WhatsAppIcon />}>Escríbenos por WhatsApp</Button>
<Button variant="secondary" size="sm">Conocer más</Button>
```

Variants: `primary` (teal, default), `accent` (warm coral for emotional CTAs), `secondary` (outlined), `ghost` (text-only), `whatsapp` (brand green). Sizes: `sm` · `md` · `lg`. Pass `block` to fill width, `href` to render an anchor.
